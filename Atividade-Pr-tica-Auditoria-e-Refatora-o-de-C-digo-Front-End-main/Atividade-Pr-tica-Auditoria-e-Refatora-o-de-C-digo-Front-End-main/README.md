# Site da Banca Central

## Autor
Kauã Douglas Pereira Moura

## Sobre o Projeto
Este projeto é uma versão refatorada do site de uma banca de revistas.  
O objetivo foi melhorar a estrutura, tornar o site mais acessível e organizado, sem mudar o conteúdo original.

## O que foi melhorado
- Estrutura semântica com `<header>`, `<nav>`, `<main>`, `<section>` e `<footer>`.  
- Menu de navegação mais acessível e fácil de usar.  
- Imagens com descrições (`alt`) corretas para leitores de tela.  
- Inclusão de link “Pular para o conteúdo” para quem navega pelo teclado.  
- Foco visível nos links para melhor experiência de navegação.  
- Layout responsivo para diferentes telas.

## Como testar
1. Abra o arquivo `index_refatorado.html` no navegador.  
2. Navegue pelo menu e confira o conteúdo da página.
